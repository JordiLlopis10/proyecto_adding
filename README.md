# Payment Gateway API

API REST desarrollada con Django Rest Framework para la gestión centralizada de pasarelas de pago.

## Tecnologías
- Python 3
- Django
- Django REST Framework
- SQLite

## Funcionalidades
- Gestión de proveedores de pago
- Registro de transacciones
- Gestión de incidencias
- Filtrado de historial
- Autenticación

## Arquitectura
El sistema sigue una arquitectura RESTful modular basada en DRF.

## Endpoints previstos
- /api/providers/
- /api/transactions/
- /api/incidents/
